# Viby Web

`web/` 是 Viby 的 Web / PWA 客户端。

它让你在手机、浏览器和 PWA 里继续控制本机 AI。

## 用户能得到什么

- 查看会话列表和聊天线程
- 实时看 AI 输出
- 发送消息、附件和重试请求
- 在会话列表里按 `进行中 / 历史` 两区浏览会话，并继续旧会话
- 记住当前设备上的输入草稿和最近一次创建偏好
- 审批权限请求
- 浏览文件、Diff 和终端
- 在手机上以 PWA 方式使用
- 创建普通会话或 manager 会话
- 在 manager 会话里集中查看任务、验收和交付状态

## 产品定位

- 移动优先
- 本地优先：AI 继续跑在你的机器上
- 实时优先：打开就能继续看会话进度
- 轻入口：页面和工作区按需加载
- 登录入口同时承担官网式产品说明：先讲清“它是什么、怎么用、为什么安全”，再进入访问面板
